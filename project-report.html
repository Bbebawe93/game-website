HTML Features

Line number is used to show the element in .html files not .php files

Head Section

Head Element
Tag: <head></head>
Location: head element is the first child element inside <html></html> tag.
Pages Element is used: Home Page, Game Page, Ranking Page, Account Page
Line Number:
Usage: includes meta data such as character encoding, page description, title and html5 shiv to support legacy browsers. This will aid web browsers render the page correctly. Also, it will help search engines display and index the page. in addition, it contains external resources such as styles, icons, web fonts and scripts
Document Type
Tag: <!doctype html>
Location: document type is declared in the first line of each page before the <html> tag.
Pages Element is used: Home Page, Game Page, Ranking Page, Account Page.
Line Number:
Usage: helps web browsers to render the page correctly including new html5 elements.
Document language
Tag: <html lang="en">
Location: document language is defined in the opening <html> tag.
Pages Element is used: Home Page, Game Page, Ranking Page, Account Page.
Line Number:
Usage: useful for search engine to translate the page to other languages. Also, it is helpful for screen readers
Character Encoding
Tag: <meta charset="utf-8">
Location: meta tag is the first child element inside the head element.
Pages Element is used: Home Page, Game Page, Ranking Page, Account Page.
Line Number:
Usage: document character encoding is set to utf-8, this will specify the character encoding of the page and support other universal languages
Page Title
Tag: <title>Page Title</title>
Location: the title tag is the second tag inside the head element.
Pages Element is used: Home Page, Game Page, Ranking Page, Account Page.
Line Number:
Usage: will de displayed in the browser tab, also used when the user bookmark the page. search engines will display page title in search engine results

Structure Elements

Structure elements were introduced in html5, they are used to add structure to html5 pages. also, they have semantic meaning that are used by browsers and search engines to identify different parts of the page such as header, main content or footer.

Header
Tag: <header></header>
Location: header is the second element inside the <body>. it is the first child of <div class="wrapper"> tag which is used to wrap header and main in one div
Pages Element is used: Home Page, Game Page, Ranking Page, Account Page.
Line Number:
Usage: used to contain introductory content to the page such as internal navigational links and logo. All the pages (Home, Game, Ranking, Account) include header element nested inside the body element
Navigation
Tag: <nav></nav>
Location: nav tag is the second element inside <div class="header-grid"> tag. which is nested inside the <header></header> tag.
Pages Element is used: Home Page, Game Page, Ranking Page, Account Page.
Line Number:
Usage: used to semantically group internal navigational links to other pages in the website.
Main
Tag: <main></main>
Location: main element is the second child element inside <div class="wrapper"> which is used to wrap header and main in one div.
<div class="wrapper"> is the first child inside <body></body>
Pages Element is used: Home Page, Game Page, Ranking Page, Account Page.
Line Number:
Usage: The main element is used indicate the main content of the page.
Article
Tag: <article></article>
Location: article element is used in Home Page. it is the first child of <div class="right-col"></div>
<div class="right-col"> is the first element nested inside the main tag.
Pages Element is used: Home Page
Line Number:
Usage: The article element is used to represent a stand alone content that still makes sense if displayed alone element
Section
Tag: <section></section>
Location: section element is used in Home Page it is the first child of <div class="left-col"></div>.
<div class="left-col"> is the second element nested inside the main tag.
Pages Element is used: Home Page
Line Number:
Usage: used to represent different sections of the page. each section starts with a heading to indicate the beginning of a section. For example:Home page has section for the Ranking table
Aside
Tag: <aside></aside>
Location : aside element is used in Ranking Page. it is the second child nested inside <main class="main-grid"></main>
Pages Element is used: Ranking page
Line Number:
Usage: The aside element is used to contain element that can be considered separated from the page such as images or ads. only Ranking page has aside element that contains images for the game
Footer
Tag: <footer></footer>
Location: footer is the last child of the <body></body> tag.
Pages Element is used: Home Page, Game Page, Ranking Page, Account Page.
Line Number:
Usage: used to contain contact information , information about the author and legal copy rights.

CSS Features

Page Design
Page Headings h1, h2, h3: CSS background color property is used to apply background color to page headings and color is used to change the font color. google fonts are used to style all page headings (h1, h2, h3), Finger Paint is the font used. Also, text align is used to set the text in the middle of the element. border radius is used to add border round the element with box shadow to improve the design
Navigation Menu: CSS display property is used to display the list items in the nav element as inline elements. also text-transform is used to capitalize the first letter of the link text. hover pseudo element selector is used to add background color and border on element hover. Fontawsone icons is used to add icons to the nav elements. padding and margin are used to add spaces around the nav links.
Classes and ID
Id : id element is used to target the logo image in the header element <img id="logo">. this is used to style the logo image and target it using its id because there are different images in the pages that has different styles applied to them.
Location : Used in all pages.
Also, <a id="current"> is used to target the current page link, this is to apply different styles to the current page which are different from other page links.
Location : Used in all pages.
Class: <img class="game-img"> is used to target images for the game in the home page. all game images has the same styles, therefore class is used to target them and apply different styles than logo image styles.
Location : Used in Home Page
Also, <class="secondary-heading"> is used to target h2 in the main element to apply different styles because there is h2 inside the footer element which has different styles.
Location : Used in Home Page
Grid Layout
the grid layout was designed using CSS3 grid layout, the whole page is designed as one grid with 1 column and 3 rows to represent header, main and footer. then header is designed as nested grid inside the main page grid. header grid has 1 row and 2 columns (1fr and 2fr). the main grid is also a nested grid inside tha main page grid. it has 1 row and 2 columns (1fr and 2fr). finally the footer grid is also a nested grid inside the main page grid it has 2 rows(2fr and 1 fr) and 1 column. Grid layout is identical in all pages

PHP Features

php functions are used to output head, header, nav and footer elements. php function avoid code duplication across website pages. php functions are stored in three php files inside php-include directory and then imported to php pages using php include statement include "path/to/php-function-file";

Head Function
File Location : located in head.php file inside php-include directory "/php-include/head.php"
Function Syntax : function output_head($page_title, $page_description){ function body } .
Usage: The head function is used to output head element to the page. The function takes two argument that are unique in each page, the argument are the page title and page description
Header Function
File Location : located in header.php file inside php-include directory."/php-include/header.php"
Function Syntax : function output_header($current_page){ function body }.
Usage: The header function is used to output header element to the page. The function takes the current page name as argument and pass it to function that outputs the navigation menu.
Navigation Function
File Location : located in header.php file inside php-include directory."/php-include/header.php"
Function Syntax : function output_navigation($current_page){ function body }.
Usage: The function is invoked inside the header function and takes current page as argument. The function has three arrays to represent page names, page links and page icons. the function loops over the arrays to output the navigation menu (unordered list), when the page name in the array matches the current page argument then id current is added to the page link. This is used by css to style current page
Footer Function
File Location : located in footer.php file inside php-include directory. "/php-include/footer.php"
Function Syntax : function output_footer() { function body }.
Usage: The footer function is used to output footer element to the page. The function uses the date object to update the copyright year information
